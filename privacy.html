Politique de Confidentialité

Notre promesse sur vos données

Mise à jour : 30 juin 2026

Vos données sont à vous

Vous nous donnez les informations de votre boutique. Notre seul travail est de les garder en sécurité pour que votre application marche bien.

Pas de publicité.

Pas de revente de vos données.

Pas de mauvaise surprise.

Ce que nous gardons

Pour créer votre compte, nous avons juste besoin de :

  • Votre numéro de téléphone — Pour vous envoyer un code de connexion par WhatsApp.
  • Votre nom — Le nom que vous voulez afficher dans l'application.
  • Votre adresse email (en option) — Si vous perdez votre numéro de téléphone ou si vous changez d'appareil, cet email vous permettra de vous reconnecter et de récupérer votre compte facilement.

Dans l'application, vous écrivez vos informations : vos produits, vos ventes, vos dépenses, vos clients et votre équipe. Nous gardons ces informations pour que vous puissiez les retrouver sur tous vos téléphones en même temps.

Marche sans internet

Patron marche très bien même quand vous n'avez pas internet. Tout ce que vous écrivez (vos ventes, vos stocks, vos calculs) reste caché et protégé dans votre téléphone.

Dès que votre téléphone retrouve internet, l'application remet tout à jour toute seule. Vos données ne quittent jamais votre téléphone sans être totalement protégées.

Vos droits

  • Vous voyez tout : Toutes vos ventes et tout votre argent sont écrits clairement dans l'application. Vous pouvez tout vérifier.
  • Vous changez tout : Vous pouvez changer le nom de votre boutique, vos prix ou vos produits quand vous voulez.
  • Vous effacez tout : Si vous voulez arrêter, vous pouvez supprimer votre compte directement dans l'application (Plus → Paramètres → Supprimer mon compte). Nous effaçons tout par terre, tout de suite et pour toujours de nos ordinateurs.
