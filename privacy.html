Politique de confidentialité

Notre position

Vos données vous appartiennent.

Patron est fait pour les commerçants. Vous nous confiez les données de votre commerce — nous les protégeons et ne les utilisons que pour faire fonctionner votre application. Pas de publicité, pas de revente, pas de surprise.

Collecte

Ce que nous stockons

Pour créer votre compte, nous demandons uniquement :

  • Votre numéro de téléphone — pour envoyer le code de vérification par SMS, rien d'autre
  • Un nom affiché — celui que vous choisissez

Dans l'application, vous créez des données : produits, ventes, dépenses, clients, fournisseurs, équipe. Ces données nous sont confiées pour les synchroniser entre vos appareils.

Si vous utilisez le chat de votre boutique ou publiez dans Le Marché (le forum communautaire), vos messages, publications et commentaires sont également stockés sur nos serveurs. Le chat de la boutique est privé : seuls vos collaborateurs le voient. Les publications du Marché sont visibles par tous les utilisateurs de Patron.

Nous ne demandons ni email, ni mot de passe, ni localisation. L'authentification biométrique (Face ID, Touch ID, empreinte digitale) fonctionne exclusivement sur votre appareil — aucune donnée biométrique ne quitte jamais votre téléphone.

Utilisation

Pourquoi ces données

  • Faire fonctionner votre compte et synchroniser vos données entre vos appareils
  • Envoyer le code de vérification par SMS à chaque connexion
  • Permettre à vos collaborateurs de voir les données selon leur rôle dans le commerce
  • Afficher vos messages dans le chat de la boutique à vos collaborateurs
  • Publier vos contributions dans Le Marché à la communauté Patron

Nous n'utilisons pas vos données pour de la publicité, du profilage ou du marketing.

Mode hors-ligne

Quand vous n'avez pas internet

Patron fonctionne sans connexion. Ventes, dépenses, créations de produits, ajustements de stock et enregistrements de paiements sont sauvegardés localement sur votre appareil dans une base chiffrée, puis synchronisés automatiquement dès que vous vous reconnectez.

Une bannière dans l'application vous indique le nombre d'opérations en attente. Si une opération ne parvient pas à se synchroniser après plusieurs tentatives, elle est archivée localement et vous en êtes informé.

Les données locales en attente de synchronisation sont protégées par un chiffrement AES-256-GCM dès leur écriture sur votre appareil. Elles ne quittent jamais votre téléphone non chiffrées.

Partenaires

Services utilisés

Nous faisons appel à deux services tiers :

  • Supabase — hébergement et base de données sécurisée
  • Twilio — envoi du SMS de vérification

Aucun autre service n'a accès à vos données.

Sécurité

Comment nous protégeons vos données

  • Toutes les communications sont chiffrées (HTTPS / TLS)
  • Votre session est stockée dans la zone sécurisée de votre appareil (Secure Enclave sur iOS, Strongbox sur Android)
  • L'authentification biométrique verrouille l'accès après quelques minutes d'inactivité
  • Les données locales hors-ligne sont chiffrées (AES-256-GCM) sur votre appareil
  • Chaque membre de votre équipe ne voit que ce que son rôle lui permet
Vos droits

Vous contrôlez tout

  • Voir vos données — tout est visible directement dans l'application
  • Modifier — votre nom, votre commerce, à tout moment
  • Supprimer — votre compte depuis l'application, immédiatement

Suppression dans l'application : Plus → Paramètres → Supprimer mon compte. Toutes vos données sont effacées immédiatement et définitivement de nos serveurs.

Si vous êtes administrateur d'un commerce avec des membres actifs, retirez-les d'abord — ou transférez votre rôle à quelqu'un d'autre.
