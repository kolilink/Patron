Politique de confidentialité

Dernière mise à jour : 28 mai 2026

🛡️

Notre engagement

Patron est une application de gestion commerciale pour les petits commerces. Vos données vous appartiennent. Nous ne les vendons pas. Nous ne les partageons jamais avec des tiers à des fins publicitaires.

Cette politique explique exactement ce que nous collectons, pourquoi, et comment vous pouvez tout supprimer à tout moment.

📋

Ce que nous collectons

Pour créer votre compte :

  • Votre numéro de téléphone WhatsApp — utilisé uniquement pour vous envoyer un code de vérification
  • Votre nom affiché (vous le choisissez librement)

Dans l'application :

  • Nom et données de votre commerce (produits, prix, stock)
  • Ventes, dépenses, créances enregistrées
  • Noms de vos clients et fournisseurs
  • Membres de votre équipe (numéro de téléphone + rôle)

Nous ne collectons pas votre email, votre mot de passe, votre localisation, ni aucune donnée biométrique. Face ID et Touch ID fonctionnent uniquement sur votre appareil — aucune donnée biométrique ne nous est envoyée.

⚙️

Comment nous utilisons vos données

  • Faire fonctionner votre compte et synchroniser vos données entre vos appareils
  • Envoyer le code de vérification WhatsApp lors de la connexion (via Twilio)
  • Permettre à vos collaborateurs de voir les données de votre commerce selon leur rôle

Nous n'utilisons pas vos données à des fins de publicité, de profilage ou d'analyse marketing.

📱

Stockage local et mode hors-ligne

Patron fonctionne sans connexion internet. Lorsque vous êtes hors-ligne, vos ventes et dépenses sont sauvegardées localement sur votre appareil dans une base de données chiffrée (SQLite).

Dès que la connexion est rétablie, ces données sont automatiquement synchronisées avec nos serveurs sécurisés.

🔗

Services tiers

Patron utilise des services tiers fiables pour faire fonctionner l'infrastructure :

  • Supabase — hébergement sécurisé de vos données (serveurs en Europe)
  • Twilio — envoi du message WhatsApp de vérification (votre numéro n'est pas utilisé à d'autres fins)

Aucun autre tiers n'a accès à vos données.

🔒

Sécurité

  • Toutes les communications entre l'application et nos serveurs sont chiffrées (HTTPS / TLS)
  • Votre session est stockée de façon sécurisée sur votre appareil (Secure Enclave sur iOS)
  • Le verrouillage biométrique (Face ID / Touch ID) protège l'accès à l'application après inactivité
  • Les accès aux données sont contrôlés par rôle — un vendeur ne voit que ce qui lui est autorisé
✅

Vos droits

Vous avez le contrôle total sur vos données :

  • Accès — toutes vos données sont visibles directement dans l'application
  • Modification — vous pouvez modifier votre nom et les données de votre commerce à tout moment
  • Suppression — vous pouvez supprimer votre compte directement depuis l'application

La suppression du compte se fait dans l'application : Paramètres → Supprimer mon compte. Toutes vos données sont supprimées immédiatement et définitivement.

Si vous êtes administrateur d'un commerce avec des membres actifs, vous devrez d'abord retirer les membres ou transférer votre rôle avant de supprimer votre compte.

🗓️

Conservation des données

Vos données sont conservées tant que votre compte est actif. Dès que vous supprimez votre compte, toutes vos données personnelles sont effacées immédiatement de nos serveurs.

Les données commerciales d'un commerce (ventes, produits, dépenses) appartiennent au commerce, pas à l'utilisateur individuel. Elles restent accessibles aux autres membres du commerce après votre départ si vous n'êtes pas l'administrateur sole.
