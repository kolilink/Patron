Politique de Confidentialité

Notre Mission : Structurer le commerce informel.

Mise à jour : 30 juin 2026

Vos données vous appartiennent

Vous nous confiez les données de votre entreprise — nous les protégeons et les utilisons uniquement pour assurer le bon fonctionnement de votre application. Pas de publicité, pas de revente, pas de surprise.

Ce que nous stockons

Pour créer votre compte, nous demandons uniquement :

  • Votre numéro de téléphone — pour envoyer votre code de vérification par WhatsApp.
  • Un nom d'affichage — celui que vous choisissez librement.

Dans l'application, vous générez des données : produits, ventes, dépenses, clients, fournisseurs et équipes. Ces données nous sont confiées afin d'être synchronisées instantanément entre tous vos appareils.

Mode hors-ligne

Patron est conçu pour fonctionner pleinement sans aucune connexion internet. Vos ventes, dépenses, créations de produits, ajustements de stocks et enregistrements de paiements sont sauvegardés localement sur votre appareil dans une base de données chiffrée, puis synchronisés automatiquement dès que vous retrouvez du réseau.

Les données locales en attente de synchronisation sont protégées par un chiffrement matériel AES-256-GCM dès leur écriture. Elles ne quittent jamais votre téléphone non chiffrées.

Vos droits

  • Droit de regard : Toutes vos données financières et commerciales sont visibles directement et en clair dans l'application.
  • Droit de modification : Vous pouvez modifier le nom de votre commerce, vos produits ou vos profils à tout moment.
  • Droit de suppression : Vous pouvez détruire votre compte directement depuis l'application (Plus → Paramètres → Supprimer mon compte). Toutes vos données seront alors effacées immédiatement, totalement et définitivement de nos serveurs.
