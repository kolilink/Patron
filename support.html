Questions fréquentes

Comment créer un commerce ?

Après votre inscription avec votre numéro WhatsApp, appuyez sur "Créer un commerce" et suivez les étapes. Votre commerce est prêt en moins d'une minute.

Comment ajouter un produit ?

Allez dans l'onglet Catalogue et appuyez sur le bouton +. Vous pouvez définir le prix d'achat, le prix de vente, le stock initial et un prix de gros si vous vendez en grande quantité.

Comment inviter un collaborateur ?

Allez dans l'onglet Plus → Équipe → générer un code d'invitation. Partagez ce code à votre collaborateur. Il l'entre dans son application pour rejoindre votre commerce. Vous choisissez son rôle : manager, vendeur ou investisseur.

Quels sont les rôles disponibles ?

Il y a quatre rôles : Administrateur (contrôle total), Manager (gestion complète sans supprimer le commerce), Vendeur (enregistre les ventes et les dépenses), Investisseur (lecture seule — voit les chiffres sans pouvoir modifier).

L'application marche sans internet ?

Oui. Ventes, dépenses, créations de produits, ajustements de stock et paiements fonctionnent hors ligne. Une bannière dans l'application indique le nombre d'opérations en attente. Tout se synchronise automatiquement dès que la connexion revient.

Comment enregistrer une vente à crédit ?

Sur l'écran de vente, activez l'option "À crédit" avant de valider. La vente apparaît dans l'historique avec le statut Crédit. Quand le client paie, allez dans la vente et appuyez sur "Enregistrer un paiement".

Qu'est-ce que Le Marché ?

Le Marché est un forum communautaire accessible depuis l'onglet Plus. Tous les utilisateurs de Patron peuvent y poser des questions, partager des conseils et discuter de commerce. Vos publications sont visibles par toute la communauté.

Comment supprimer mon compte ?

Allez dans Plus → Paramètres → Supprimer mon compte. Toutes vos données sont effacées immédiatement et définitivement. Si vous êtes administrateur d'un commerce avec d'autres membres actifs, retirez-les d'abord.

Je n'arrive pas à me connecter.

La connexion se fait par WhatsApp : vous recevez un code sur votre numéro et vous le copiez dans l'application. Si le code n'arrive pas après quelques minutes, vérifiez que votre numéro est bien au format international (ex. +224...) et réessayez. Si le problème persiste, écrivez-nous sur WhatsApp.
